# 檢查雲端基礎結構上Adobe Commerce的磁碟空間限制

若要檢查您在雲端基礎結構環境中每個Adobe Commerce有多少空間：

1. 登入[雲端主控台](https://console.adobecommerce.com/)。
2. 在&#x200B;**所有專案**&#x200B;儀表板上，選取相關的專案。 在下角，檢視磁碟空間可用性。

[專案空間](/help/assets/disk-space.png)
