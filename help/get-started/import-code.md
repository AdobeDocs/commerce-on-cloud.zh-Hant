# 匯入現有的Commerce程式碼

Adobe建議您從空白範本或匯入現有程式碼，在雲端基礎結構專案上建立Commerce。 我們建議先從空白範本開始，然後將現有程式碼匯入新專案。 這可確保您擁有支援專案的所有必要檔案。

匯入現有程式碼的完整工作流程包括下列步驟：

- 取代範本程式碼
- 匯入資料庫和媒體內容
- 清除快取並驗證匯入
